## 🧠 数学原理

**洛伦兹系统**（1963，Edward Lorenz）— 混沌理论的奠基模型：

```
dx/dt = σ(y - x)       σ = 10  (普朗特数)
dy/dt = x(ρ - z) - y    ρ = 28  (瑞利数)
dz/dt = xy - βz        β = 8/3 (几何因子)
```

这个简单的三维微分方程组产生了著名的**蝴蝶形吸引子**——初始条件的微小差异会被指数级放大，即"蝴蝶效应"。

## ✨ 实现细节

| 特性 | 说明 |
|------|------|
| 🧮 **积分方法** | **RK4 四阶龙格库塔**，步长 0.004，精度高 |
| 🎨 **12 条轨迹** | 初始条件 (1,1,1) 附近微扰 0.05，展示**蝴蝶效应** |
| 📐 **每轨迹 2500 点** | 循环缓冲区，旧数据被新数据覆盖 |
| 🌈 **颜色流动** | HSL 色相随时间推移，旧点暗新点亮 |
| 💡 **头部光点** | 球体追踪每条轨迹当前位置 |
